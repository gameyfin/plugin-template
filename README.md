# plugin-template (WIP)
Template project for creating a new Gameyfin plugin.
This template provides a basic structure for creating a Gameyfin plugin, including the necessary dependencies and configuration files.
The plugin system of Gameyfin is based on [PF4J](https://github.com/pf4j/pf4j).

See also the official [Gameyfin Plugin Development Guide](https://gameyfin.github.io/docs/plugins/) for more information.
